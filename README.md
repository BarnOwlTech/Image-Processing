# Image-Processing
Лабораторные работы по "Обработке Изображений".

1 Лабораторная Работа:
- Цветовая коррекция изображений.
- Коррекция с опорным цветом.
-	Серый мир.
-	По виду функции преобразования.
-	Яркостная коррекция в интерактивном режиме по виду функции преобразования (необязательное дополнительное задание).
-	График функции кусочно линейный.
- График функции интерполируется сплайном.
- Коррекция на основе гистограммы.
-	Нормализация гистограммы.
-	Эквализация гистограммы.

2 Лабораторная Работа:
- Бинаризация с ручным выбором порога (верхняя, нижняя и по диапазону).
- Бинаризация методом Otsu глобальная.
- Бинаризация методом Otsu локальная (с предварительным разбиением на фрагменты - по каждому своя бинаризация).
- Бинаризация методом Otsu иерархическая (выполняется с иерархическим разбиением каждой уже полученной области, пока возможно разбивать).
- Квантование по яркости (с ручным заданием количества уровней).

3 Лабораторная Работа:
- Морфологические преобразования
- Следует реализовать выделение проблемных зубьев у шестеренки. Пример разобран на лекции.
- Описание есть у Стокмана.
- Использовать только морфологические операции для бинарных изображений.
- Erode – размывание (операция сужения).
- Dilate – растягивание (операция расширения).

4 Лабораторная Работа:
- Улучшение изображения с помощью частотных фильтров.
- Идеальный фильтр, Батерворта и Гауссиан.

5 Лабораторная Работа:
- Применение частотных фильтров для сравнения их.
- Идеальный фильтр, Батерворта и Гауссиан.
